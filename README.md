# NFL Skill Player Data Explorer

Temporary GitHub Pages prototype for exploring 2015-2024 nflverse regular-season QB/RB/WR/TE player-game data.

## Features

- Static GitHub Pages deployment: no server required.
- Data stored behind the UI as JSON files in `data/`.
- Filters for season, position, team, player name, and minimum PPR points.
- Sorting by common stat columns.
- Summary cards for matched rows, unique players, average PPR, and top row.
- Column picker to create small custom tables.
- Download visible rows as CSV.

## Dataset

- Source: nflverse-data GitHub releases.
- Scope: 2015-2024 regular seasons.
- Positions: QB, RB, WR, TE.
- Row count: 55,909 player-game rows.

This is intended for personal exploration/prototyping. nflverse code is open source; underlying NFL data belongs to respective owners and is governed by their terms.
